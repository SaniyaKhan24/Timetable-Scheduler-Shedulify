# Automatic Timetable Scheduling System

AI-powered timetable generation using Genetic Algorithms with constraint satisfaction.

## Features

- **Genetic Algorithm**: Population-based optimization with selection, crossover, and mutation
- **Constraint Handling**: Backtracking and Branch & Bound for conflict resolution
- **Fitness Evaluation**: Multi-objective scoring for schedule quality
- **Web Interface**: Simple HTML/CSS/JS frontend
- **RESTful API**: Flask backend with JSON communication

## Setup

1. Install dependencies:
